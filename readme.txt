== Description ==

Access a library of premade responsive templates and use custom widgets create the website of your dreams lightning fast. Amaze your website visitors with scrolling animations, professional designs, and unique layouts with ThemeShark Templates & Widgets for Elementor.

== Installation ==

= Minimum Requirements =

* WordPress 5.2 or greater
* PHP version 7.0 or greater
* MySQL version 5.0 or greater
* Elementor version 3.0 or greater

= Installation =

1. If you haven't already, Install and Activate [Elementor Page Builder](https://elementor.com/). You can find it by searching for it on the WordPress plugins directory (Plugins > Add New).
2. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
3. Activate the plugin through the 'Plugins' menu in WordPress.
